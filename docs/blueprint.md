# **App Name**: Aprende Con Ritmo

## Core Features:

- User Authentication: Secure user authentication system supporting email/password registration and login. Access to features and content will be gated based on assigned role.
- Dashboard: Role-based dashboards for students/parents, teachers, and administrators. Displaying relevant information, quick actions, and summarized data.
- Class Scheduling: Allow students/parents to schedule classes and teachers to manage their availability.
- Resource Library: Organized repository of educational materials, filterable by instrument and accessible to all users.
- Progress Tracking: Teachers can input student progress which can then be viewed by student/parent accounts.
- User Management: Admin interface to manage user accounts (teachers and students), roles, and permissions.

## Style Guidelines:

- Primary color: Warm Beige (#F5F5DC) to create a welcoming and comfortable atmosphere.
- Secondary color: Soft Peach (#FFE5B4) to evoke feelings of joy and creativity.
- Accent color: Muted Coral (#FF8B7A) to highlight important elements and calls to action.
- Background color: Off-White (#FAFAFA) to provide a clean and neutral backdrop.
- Body and headline font: 'Nunito Sans' (sans-serif) for clear, readable text with a friendly feel.
- Card-based layout to organize information on dashboards.
- Consistent and clear icons for navigation and features.
- Subtle transitions and animations for feedback and engagement.
